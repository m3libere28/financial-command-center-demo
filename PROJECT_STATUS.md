# Financial Command Center Demo — Project Status

Last updated: 2026-07-23

## Repository role

Standalone public product demo using the fictional Morgan household.

## Current state

- Runtime is static HTML, CSS, and JavaScript.
- Data is fictional and centralized in `data.js`.
- Copilot responses are scripted educational demo content.
- No Supabase, production API, analytics, tracking, form submission, or persistent storage is used.
- `.github/workflows/pages.yml` is the sole allowlisted Pages workflow.

## Release boundary

Pull requests do not deploy. Merge, release, Pages settings, visibility, data collection, and external integrations require explicit human approval.
